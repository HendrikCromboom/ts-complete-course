# TypeScript Complete Course

### How to install

* Clone or fork this repository
* In terminal go to the designated folder
    * npm init
    * npm install --save-dev lite-server
* In the json file created by npm add a new line to the scripts section:
    * "start": "lite-server"
   
*To launch: npm start
